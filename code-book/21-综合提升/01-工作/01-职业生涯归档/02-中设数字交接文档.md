### 后端

```java
/**
 * 子表数据：
 * 0-集采协议和历史合同都不存在；
 * 1-存在集采协议；（用于新增集采价时对已有价格对物料置灰）
 * 2-存在历史合同；
 * 3-集采协议和历史合同都存在
 */
private int subData;
```

通过标签系统的思想为每个物料打上价格标签，方便扩展。

通过**位运算**实现标签的增删查：

- 添加标签：或运算
- 删除标签：取反之后与运算
- 查询标签：`(tagInfo & matchTag) == matchTag`
