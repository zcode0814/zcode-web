### 红黑树特点

- 根节点和叶子结点是黑色
- 每个红色节点的两个子节点一定都是黑色
- **从任一节点到其子树中每个叶子节点的路径都包含相同数量的黑色节点**；

### 红黑树保持平衡

### 红黑树和 AVL 树的对比

AVL 树通过严格的平衡条件来确保树的平衡，即任意节点的左右子树高度差不超过 1。因此，AVL 树的平衡性更加严格，可能需要更多的旋转操作来维持平衡。

AVL 树适用于对读取操作频繁的场景，因为它提供了更快的查找性能。

红黑树适用于对插入和删除操作频繁的场景，因为它提供了更好的平衡性能。

```java
public static void main(String[] args) {
        // 自定义比较器，按照键的长度进行排序
        Comparator<String> customComparator = Comparator.comparingInt(String::length);

        // 使用自定义比较器创建TreeMap

        // 使用自定义比较器创建TreeMap
        TreeMap<String, Integer> treeMap = new TreeMap<>();

        // 向TreeMap中添加元素
        treeMap.put("apple", 1);
        treeMap.put("banana", 2);
        treeMap.put("orange", 3);

        // 遍历输出TreeMap
        for (String key : treeMap.keySet()) {
            System.out.println(key + ": " + treeMap.get(key));
        }

        // 创建一个线程安全的TreeMap
        Map<String, Integer> synchronizedTreeMap = Collections.synchronizedSortedMap(new TreeMap<>());

        // 向线程安全的TreeMap中添加元素
        synchronizedTreeMap.put("apple", 1);
        synchronizedTreeMap.put("banana", 2);
        synchronizedTreeMap.put("orange", 3);

        // 遍历输出线程安全的TreeMap
        synchronized (synchronizedTreeMap) {
            for (String key : synchronizedTreeMap.keySet()) {
                System.out.println(key + ": " + synchronizedTreeMap.get(key));
            }
        }
    }
```
