### 解题模板

1. 数组

```java
class Solution {
    public int rob(int[] nums) {
        return rob1(nums, 0, nums.length);
    }
    // 模板
    private int rob1(int[] nums, int start, int end) { // [start,end) 左闭右开
        int f0 = 0, f1 = 0;
        for (int i = start; i < end; i++) {
            int newF = Math.max(f1, f0 + nums[i]);
            f0 = f1;
            f1 = newF;
        }
        return f1;
    }
}
```
