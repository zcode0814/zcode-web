## 多行合并一行

**方案一**

1. 全选文本
2. `command+shift+L`在每一行的最后面生成光标
3. 右方向键`——>`将光标移动到每一行的首部，再按删除键，删除换行符
4. 多行合并为紧凑的一行，如果需要增加间隔或符号，再按空格或对应符号即可

![](https://yitiaoit.oss-cn-beijing.aliyuncs.com/img/image-20240327100719141.png)

**方案二**

1. 全选文本
2. `command+shift+J`快速合并为一行，但是会有空格分隔

![](https://yitiaoit.oss-cn-beijing.aliyuncs.com/img/image-20240327100844519.png)

## 多行选取

将下方文本中的`. `修改为`-`，方法如下：

- 如果行数不多，按住 command 即可同时选取多行，删除后再输入即可。
- 如果行数很多，可以 command+option+F 全局替换。
