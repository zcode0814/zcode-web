## 连接超时

```
The last packet successfully received from the server was xxx milliseconds ago
```

产生的原因：应用方的数据库连接有效期时间，大于数据库自己设置的有效期。即连接池内的连接已经过期，但是没有被剔除。

解决思路无非是增加过期时间，或者检测空闲的连接或者将连接池的连接有效时间设置的小于数据库的有效时间。
